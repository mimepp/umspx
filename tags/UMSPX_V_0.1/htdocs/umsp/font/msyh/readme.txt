1. how to find a msyh.ttf file
use google to search:
+"msyh.ttf" +"index of"

2. copy the link to msyh_download script

3. run the msyh_download script
* for windows:
 * please double-click on msyh_download.bat to run it
* for linux:
 * please use "sh msyh_download.sh" to run it

4. wait for the finishing of download, you will get the font

5. please note: copyright of msyh.ttf is Microsoft.
